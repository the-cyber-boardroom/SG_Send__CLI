## Changes Planned

### 1. New Safe Types to Create

| New Type | Purpose | Pattern |
|----------|---------|---------|
| `Safe_UInt__Key_Size` | RSA key size (e.g. 4096) | `max_value = 16384` |
| `Safe_UInt__Lock_Timeout` | Auto-lock timeout in seconds | `max_value = 86400` (24h) |
| `Safe_Str__Content_Type` | MIME content type (e.g. `application/octet-stream`) | Regex: `[^a-zA-Z0-9/._+\-]`, max 256 |
| `Safe_Str__PEM_Key` | PEM-encoded public key data | Regex: printable + whitespace, max 8192 |
