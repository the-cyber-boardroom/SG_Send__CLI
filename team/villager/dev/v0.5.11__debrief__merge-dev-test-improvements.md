### The Pattern: Real Lambda Test Server

The key technique (documented in `team/humans/dinis_cruz/briefs/03/14/test-send-lambda/`) is:

```python
# conftest.py — session-scoped real HTTP server
from sgraph_ai_app_send.lambda__user.testing.Send__User_Lambda__Test_Server import setup__send_user_lambda__test_server

@pytest.fixture(scope='session')
def send_server():
    with setup__send_user_lambda__test_server() as test_objs:
        yield test_objs  # .server_url, .access_token, .write_key
```

This replaces: env vars, `.env` file loading, `@pytest.mark.skipif`, subprocess server management, and mock patches — all at once.
