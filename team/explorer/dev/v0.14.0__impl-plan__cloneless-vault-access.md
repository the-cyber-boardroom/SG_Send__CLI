## Phase 1: Vault__Remote Core (read-only)

### File: `sg_send_cli/sync/Vault__Remote.py`

```python
class Vault__Remote(Type_Safe):
    crypto : Vault__Crypto
    api    : Vault__API

    def _resolve_keys(self, vault_key: str) -> dict:
        """Derive all keys from vault_key. Returns keys dict."""

    def _fetch_tree(self, vault_key: str) -> tuple[dict, dict]:
        """Fetch and decrypt tree.json + settings.json.
        Returns (tree_data, settings_data).
        Cost: 2 API calls."""

    def _flatten_tree(self, tree: dict, prefix: str = '') -> dict:
        """Flatten nested tree into {path: {file_id, size, ...}}.
        NOTE: Reuse Vault__Sync._flatten_tree logic — consider
        extracting to a shared helper class."""

    def list_files(self, vault_key: str) -> dict:
        """Returns {files: [{path, size, file_id}...], vault_name, version, file_count, total_size}"""

    def read_file(self, vault_key: str, file_path: str) -> bytes:
        """Fetch tree, find file_id for path, download + decrypt.
        Raises RuntimeError if path not found."""

    def read_file_by_id(self, vault_key: str, file_id: str) -> bytes:
        """Direct download + decrypt by file_id. Fastest path.
        Skips tree fetch entirely."""

    def save_file(self, vault_key: str, file_path: str, dest: str) -> str:
        """read_file + write to dest. Returns absolute path written."""

    def get_tree(self, vault_key: str) -> dict:
        """Return full tree_data (for callers who want to cache it)."""

    def get_info(self, vault_key: str) -> dict:
        """Return vault summary: name, version, file_count, total_size, updated."""
```

### Key Implementation Notes

1. **_flatten_tree duplication** — The same logic exists in `Vault__Sync._flatten_tree`. Extract into a standalone `Vault__Tree_Utils` class or have `Vault__Remote` depend on the shared method. Per Type_Safe rules, no module-level functions — put it on a class.

2. **Error handling** — `read_file` must give a clear error when path doesn't match:
   ```
   RuntimeError: File 'configs/missing.json' not found in vault.
   Available files: configs/EC2.json, configs/deploy.json, ...
   ```

3. **No local state** — No .sg_vault directory, no object store, no refs. Pure API + crypto.

## Phase 2: CLI Commands

### File: `sg_send_cli/cli/CLI__Vault.py` (additions)

#### `sg-send-cli ls <vault_key>`

```python
def cmd_ls(self, args):
    remote = self._create_remote(args.base_url)
    result = remote.list_files(args.vault_key)
    print(f'📦 {result["vault_name"]} (version {result["version"]})')
    print(f'📂 {result["file_count"]} files, {self._format_size(result["total_size"])}')
    print()
    for f in result['files']:
        print(f'  {self._format_size(f["size"]):>8}  {f["path"]}')
```

#### `sg-send-cli cat <vault_key> <path>`

```python
def cmd_cat(self, args):
    remote = self._create_remote(args.base_url)
    data   = remote.read_file(args.vault_key, args.file_path)
    sys.stdout.buffer.write(data)  # binary-safe for piping
```

#### `sg-send-cli get <vault_key> <path> [dest]`

```python
def cmd_get(self, args):
    remote = self._create_remote(args.base_url)
    dest   = args.dest or os.path.basename(args.file_path)
    path   = remote.save_file(args.vault_key, args.file_path, dest)
    size   = os.path.getsize(path)
    print(f'📥 Saved {args.file_path} → {path} ({self._format_size(size)})')
```

#### `sg-send-cli info <vault_key>`

```python
def cmd_info(self, args):
```

## Lambda Integration Example

Once implemented, a Lambda function can read a vault file in ~2 API calls:

```python
from sg_send_cli.sync.Vault__Remote import Vault__Remote
from sg_send_cli.crypto.Vault__Crypto import Vault__Crypto
from sg_send_cli.api.Vault__API import Vault__API

def lambda_handler(event, context):
    remote = Vault__Remote(
        crypto = Vault__Crypto(),
        api    = Vault__API(base_url='https://send.sgraph.ai').setup()
    )
    vault_key = event['vault_key']
    config    = remote.read_file(vault_key, 'configs/EC2.json')
    return json.loads(config)
```

**Cold start impact:** ~0.8s instead of ~13s.
